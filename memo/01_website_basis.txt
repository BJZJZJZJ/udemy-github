1. 웹사이트 동작방식
     * 자세하게 들어가려면 IP (Internet Protocol) 공부
    1) 클라이언트 -> DNS -> : 브라우저를 통해 도메인에 해당하는 IP 주소 요청(Request), 응답(Response)
    2) 클라이언트 -> 서버 : DNS로부터 받은 IP 주소로 Request
    3) 서버 -> 클라이언트 : 받은 요청에 대한 Response


2. 서버로부터 Request를 받았을 때,
     <link> 태그로 엮인 파일들은 병함된게 아니라
     html 파일을 받은 후 추가로 요청해서 받게 됨ㄴ
